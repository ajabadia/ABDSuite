# Web Interface Audit: Aseptic Standard

Este skill se activa para realizar verificaciones de calidad sobre la interfaz final entregada.

## ⚡ Cuándo activar
- En la fase de Cierre y Verificación (Walkthrough) de cualquier tarea de UI.
- Tras refactorizaciones visuales o cambios en el sistema de temas.

## 🛠️ Checklist de Auditoría
1. **Accesibilidad (A11y)**: ¿Es navegable por teclado? ¿Tiene el contraste adecuado?
2. **UX Industrial**: ¿El flujo de trabajo es lógico? ¿Los tiempos de respuesta son aceptables?
3. **Consistencia Visual**: ¿Sigue estrictamente el estándar `uncodixfy`?
4. **Respuesta Técnica**: ¿Los mensajes de error son claros y técnicos?

## 📊 Criterios de Éxito
- Puntuación de contraste > 4.5:1.
- Cero errores de consola en el flujo principal.
- Feedback visual instantáneo en todas las acciones críticas.
